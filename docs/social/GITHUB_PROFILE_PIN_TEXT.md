# GitHub Profile Pin Text

PedsCore is an open-source pediatric and neonatal clinical tools library with evidence traceability, WHO Growth charts and no clinical data storage.

Public alpha:

- 81 cataloged tools.
- 21 fully implemented tools.
- 4 partially implemented WHO Growth entries.
- MIT source code.
- Vercel production web app with aggregate public stats API.
