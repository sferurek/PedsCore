# PedsCore v0.1.0-alpha

## Summary

PedsCore v0.1.0-alpha is the first public alpha release of the project. It provides a bilingual, static, open-source pediatric and neonatal clinical tools library with a catalog-driven web interface, evidence traceability, dynamic forms, initial validated calculators and informational clinical rules.

Public web URL:

https://sferurek.github.io/PedsCore/

## Implemented Tools

The alpha release includes 17 implemented tools:

- Apgar.
- Silverman-Andersen.
- FLACC.
- QTc Bazett.
- QTc Fridericia.
- QTc Framingham.
- QTc Hodges.
- Bedside Schwartz.
- Westley Croup Score.
- PRAM.
- Clinical Dehydration Scale.
- PECARN TBI under 2 years.
- PECARN TBI 2 years or older.
- CATCH.
- CHALICE.
- SIPA.
- NIPS.

## Tools In Validation

Several cataloged tools remain in validation because they require one or more of:

- Primary source confirmation.
- Complete scoring table.
- Exact variant selection.
- Published interpretation bands or cut-offs.
- Licensing or copyright review.
- Expert clinical review.

Examples include PEWS variants, Wood-Downes-Ferres, Pediatric Glasgow Coma Scale, CRIES, CHEOPS, Thompson HIE, PIPP/PIPP-R, COMFORTneo, Finnegan, growth percentiles, intensive care scores and resuscitation algorithms.

## Catalog-only And License-sensitive Tools

Some tools are intentionally catalog-only or blocked because of licensing, copyright, regulatory risk or maintainer decision requirements. These include, among others, Wong-Baker Faces, Bayley, Denver II, Orbegozo-related data review, some proprietary or license-sensitive screening tools, and high-risk official algorithms.

## Scope

This alpha release includes:

- 78 cataloged pediatric and neonatal tools.
- ES/EN web interface.
- Catalog search, category pages and individual tool pages.
- Dynamic forms for implemented tools.
- Result panels with traceability.
- Visible references, validation status and disclaimers.
- Evidence documentation under `docs/evidence`.
- GitHub Pages deployment.

## Limitations

- Alpha release: content, wording and UX are still under active review.
- Not every cataloged tool has active calculation.
- Some Spanish clinical wording is not yet validated as literal Spain Spanish source wording.
- No backend, account system, analytics or clinical data persistence exists.
- No toxicology tools are included in this phase.
- Some implemented tools still require ongoing source wording, licensing and expert review before broader clinical use.

## Disclaimer

PedsCore is an educational and informational reference for healthcare professionals. It does not replace clinical judgment, local protocols, institutional requirements, specialist consultation or individual patient assessment. It must not be used as the sole basis for clinical decision-making.

PedsCore does not store patient data.
