# Website Copy (ES/EN)

This document contains suggested text snippets for the PedsCore web interface in both English and Spanish. These can be used when implementing the user interface.

## Homepage / Página principal

### English

- **Title:** PedsCore
- **Subtitle:** Open‑source paediatric and neonatal clinical tools
- **Search placeholder:** "Search for a tool, score or scale…"
- **Call to action:** "Select a tool to begin"

### Español

- **Título:** PedsCore
- **Subtítulo:** Herramientas clínicas pediátricas y neonatales de código abierto
- **Campo de búsqueda:** "Buscar herramienta, score o escala…"

## About page / Página sobre

### English

PedsCore provides clinicians and researchers with validated paediatric and neonatal tools. Our mission is to democratise access to critical clinical calculators and scores. All tools are based on primary literature and guidelines.

### Español

PedsCore ofrece a los clínicos e investigadores herramientas pediátricas y neonatales validadas. Nuestra misión es democratizar el acceso a calculadoras y scores clínicos esenciales. Todas las herramientas se basan en literatura primaria y guías oficiales.

## Disclaimer page / Página de aviso legal

### English

PedsCore is an educational resource. It is not medical advice. Clinicians must use their judgement and follow local protocols. See the full disclaimer for details.

### Español

PedsCore es un recurso educativo. No es asesoramiento médico. Los profesionales deben emplear su criterio y seguir los protocolos locales. Consulta el aviso legal completo para más detalles.

## Contribute page / Página de contribución

### English

We welcome contributions! Use the issue templates on GitHub to suggest new tools, report bugs or improve documentation. See our contributing guidelines for details.

### Español

¡Las contribuciones son bienvenidas! Utiliza las plantillas de issues en GitHub para sugerir nuevas herramientas, reportar errores o mejorar la documentación. Consulta nuestras directrices de contribución para más detalles.
